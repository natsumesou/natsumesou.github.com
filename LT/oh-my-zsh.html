oh-my-zshのススメ

みなさんはbash派ですか？zsh派ですか？
自分はzsh派です
毎日触れる開発環境、メンテナンスできていますか？
zshのイケてる設定知りたいですよね？

そこで oh-my-zsh の出番です！！

豊富なプロンプトのテーマ(100以上!!) * お洒落に気を取られすぎてプロンプトが重くなってしまうテーマもあるので注意

豊富なプラグイン

便利なaliasや関数たち
  • git
  • svn
  • osx
  • bundler
  • github
など
週に一回自動でアップデートをしてくれる！（詳細は後で）
導入方法 
curl -L https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh | sh
ディレクトリ構成(重要なところだけ) 
        ~/.zshrc       -> oh-my-zsh用の.zshrc テーマやプラグインの設定
        ~/.oh-my-zsh
            |-custom       -> 個々の.zshrcを *.zsh としておいておけば
                                        最後に読みこんでくれる
            |-lib          -> oh-my-zshの基本的な設定
            |-plugins      -> プラグイン
            |-themes       -> プロンプトのテーマ

テーマのカスタマイズ

~/.zshrc で 
ZSH_THEME="mrtazz"
のように自分の好みのテーマファイル名を入力するだけで素敵なプロンプトの完成
イカしたテーマだけでなく実用的なプロンプトで開発効率アップ！

プラグインのカスタマイズ

~/.zshrc で 
plugins=(git github osx svn)
のように取り込みたいプラグインを列挙するだけ！

自動アップデート

一週間ごとにgithubでmasterに取り込まれた変更を自動で更新してくれる！(git pull origin master するだけ)
oh-my-zshはgithubで開発中(watched:5680 , forked: 2138 [2012/6/13現在])
みんなも是非pull requestを送って便利な機能を取り込んでもらおう！
でもpull requestが取り込まれるのに結構時間かかる
