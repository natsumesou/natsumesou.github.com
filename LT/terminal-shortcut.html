ターミナル生活

みなさんは一日どれだけターミナルの画面に触れていますか？
ファイル操作・Git・cap deploy・ssh etc...
いろんな操作をターミナルを介して行なっています
そんなターミナルをちゃんと使いこなせていますか？ 
mb ~/hoge ~/fuba/hoge
git checkout miss origin/dev1 dev1
先頭文字のタイポしたり
途中にいらない文字を入れてしまったり...
ターミナルでの操作で便利なキーボードショートカットを利用しよう！
ターミナルの2種類のキーバインド
  • 端末のキーバインド(stty)
  • shellのキーバインド(bash/zsh)
端末のキーバインド 
stty -a
cchars: discard = ^O; dsusp = ^Y; eof = ^D; eol = ;
eol2 = ; erase = ^?; intr = ^C; kill = ^U; lnext = ^V;
min = 1; quit = ^\; reprint = ^R; start = ^Q; status = ^T;
stop = ^S; susp = ^Z; time = 0; werase = ^W;
  • intr = ^C; プロセス終了の割り込みシグナルを送る
  • werase = ^W; 直前の単語を削除
など 詳しくは 
man stty
shellのキーバインド(zsh) bashもzshも基本的にemacsのキーバインドが適用されている 
bash$  bind -p
zsh$  bindkey
"^A" beginning-of-line
"^B" backward-char
"^D" delete-char-or-list
...
  • ^A 行頭に移動
  • ^E 行末に移動
  • ^N 次の履歴を表示
  • ^P 前の履歴を表示
  • ^W 直前の一単語を削除
  • ^L ターミナルをクリアする
  • ^S 履歴の前方インクリメタルサーチ
  • ^R 履歴の後方インクリメタルサーチ
これだけでもターミナル上での作業が楽に！
emacsキーバインドだけでなくviのキーバインドも使える 
bash$  set -o vi
zsh$  bindkey -v
一回やってみたら効率落ちたのでヤメました やっぱターミナル上はemacsキーバインドのほうがイイ！
